This is some sample content. Scroll down to see the navigation menu change its appearance. The menu has a fixed position and will always be visible at the top of the viewport, providing easy access to all sections of the site. This fixed positioning is crucial for user experience on long-scrolling pages.

The **`fixed`** CSS position property ensures the element stays in place relative to the browser window. We also add padding to the body to prevent the content from being hidden behind the menu. The **`z-index`** property ensures the menu appears on top of other content.

As you scroll, a JavaScript function listens for the **`scroll`** event. Once the scroll position passes a specific threshold, it adds a new class, **`.scrolled`**, to the navigation bar element. This class contains the new styles for the background color, font color, and other visual changes. This is a common and effective technique for creating dynamic headers.

The **`:hover`** pseudo-class in CSS handles the mouse-over effect on each link. This provides immediate, non-intrusive feedback to the user, enhancing the interactivity and overall feel of the menu. Together, these techniques create a professional and responsive user interface.
